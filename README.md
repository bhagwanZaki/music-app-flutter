# Music Lyrics App

Music Lyrics App app provide lyrics of tracksusing MusicMix api services. The App is created on the basis of <b>Bloc pattern.</b>

This app consist of 3 pages
<ol>
<li>Splash Screen</li>
<li>Music List Screen</li>
<li>Music Lyrics Screen</li>
</ol>
